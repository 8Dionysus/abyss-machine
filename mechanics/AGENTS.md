# mechanics Route

`mechanics/` owns durable host-machine movement packages. Add or edit a package
when the work has a recurring trigger, clear owner boundary, inputs, outputs,
and validation route.

Each package keeps:

- `AGENTS.md`
- `README.md`
- `DIRECTION.md`
- `PROVENANCE.md`
- `PARTS.md`
- `ROADMAP.md`
- `LANDING_LOG.md`
- `docs/README.md`
- `parts/README.md`

Do not use mechanics as a dumping ground for generic notes.
